# Mridangam Stroke Detection

This project uses a Convolutional Neural Network (CNN) to classify different strokes (syllables) of the Mridangam, a South Indian percussion instrument, from audio files. It includes scripts for training the model and a Flask web application for real-time prediction.

### рџ“„ File Descriptions

*   **`training the model.py`**: This script loads the audio dataset, extracts MFCC features, builds the CNN architecture, and trains the model. The trained model is saved as `audio_classification_model.h5`.
*   **`prediction.py`**: An example script demonstrating how to load the trained model and predict the stroke from a single audio file.
*   **`app.py`**: A Flask web server that provides a simple web interface to upload an audio file and view the predicted Mridangam stroke.
*   **`templates/index.html`**: The HTML file that renders the user interface for the web application. (You will need to create this file).
*   **`requirements.txt`**: A list of all Python packages required to run the project.
*   **`audio_classification_model.h5`**: The pre-trained model file generated by `training the model.py`.
